# Snowfall  | medium | CRYPTO

## Описание

Все просто. Необходимо расшифровать флаг. Флаг шифруется на сервере.

## Solution
Ключевая ошибка алгоритма заключается в том, что LFSR и FSM не были инициализированы отдельно. Как результат, ко второму циклу алгоритма мы можем восстановить и LFSR и FSM брутфорсом только ключ[0]. Это означает, что инициализация FSM_R2 в этой реализации алгоритма была фактически ненужна.

  ```cpp
  LFSR[0] = key_0_init;
  for (int i = 0; i < 15; ++i){
      LFSR[i + 1] = (LFSR[i] >> (1)) + ((((LFSR[i] >> 15)) ^ (LFSR[i] & 0x1)) << 16);
  }
  uint16_t FSM_R1_0 = 0xFFFF - LFSR[15];
  uint16_t FSM_R1_1 = LFSR[15];
  uint16_t FSM_R2_1 = (SBOX[(0xFF00 & FSM_R1_0) >> 8] << 8) + SBOX[0x00FF & FSM_R1_0];
  ```
        
Как только состояние LFSR и FSM во втором цикле определено, мы можем восстановить весь поток ключей и расшифровать флаг, следуя обычному процессу дешифровки.
